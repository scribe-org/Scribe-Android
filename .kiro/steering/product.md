# Scribe-Android — Product Overview

Scribe-Android is an open-source Android keyboard app for language learners. It provides language-specific keyboards with features like translation (beta), verb conjugation, and word annotation to help users communicate with confidence.

## Supported Languages
German (DE), English (EN), Spanish (ES), French (FR), Italian (IT), Portuguese (PT), Russian (RU), Swedish (SV)

## Key Characteristics
- Fully offline — all language data is bundled in-app (sourced from Wikidata via Scribe-Data)
- No usage data collection, no system access beyond keyboard functionality
- WIP port of Scribe-iOS; currently implements base keyboard functionality
- Part of the broader Scribe ecosystem: Scribe-iOS, Scribe-Data, Scribe-i18n

## Current Features
- Dynamic layouts for cross-device performance
- Auto-capitalization after `.`, `?`, `!`
- Hold-to-select characters, key pop-up views
- Symbol/number input with auto-return to letters on space
- Dark/light mode support

## Not Yet Supported
- Floating keyboard mode
- Full feature parity with Scribe-iOS
